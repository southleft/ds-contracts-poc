# System memory

What this design system is and is for — the standing context every generating
surface reads before producing anything. Edit freely (or via Contract Hub's
Context page); this ships inside the catalog.

- This system serves data-dense B2B product surfaces: settings, directories,
  billing, member management.
- Tone: quiet and information-first. Color signals state, never decoration.
- Density matters: tables and forms should hold up at compact rhythm.
- When a request needs something the catalog lacks, report the gap — never
  approximate with lookalikes.
